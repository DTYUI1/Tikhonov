# Программа для удаления содержимого в скобках

Программа обрабатывает строки, содержащие вложенные скобки, удаляя все содержимое внутри скобок (включая сами скобки) и оставляя только символы, находящиеся за пределами скобок.

## Описание алгоритма

Программа анализирует строку символ за символом, отслеживая баланс открывающих `(` и закрывающих `)` скобок. Символы добавляются в результирующую строку только если они находятся вне всех скобок (когда счетчик открытых скобок равен счетчику закрытых).

**Пример работы:**
- Входная строка: `"a(bc(de)f)g(h(i))"`
- Выходная строка: `"ag"`
- Объяснение: Удалены все символы внутри скобок, включая вложенные скобки

## Особенности реализации

- Поддерживает произвольный уровень вложенности скобок
- Выполняет проверку корректности расстановки скобок
- Выводит информативные сообщения об ошибках при:
  - Лишних закрывающих скобках
  - Незакрытых скобках
- Обрабатывает символы, которые не являются скобками

## Проверка корректности скобок

Программа включает два типа проверок:

1. **Лишние закрывающие скобки** - если встретилась закрывающая скобка без соответствующей открывающей
2. **Незакрытые скобки** - если после обработки строки количество открывающих и закрывающих скобок не совпадает

## Примеры работы

### Пример 1: Корректная строка
```
Введите данные: a(bc(de)f)g(h(i))
Результат: ag
```

### Пример 2: Ошибка - лишняя закрывающая скобка
```
Введите данные: a(b)c)
Ошибка: лишняя закрывающая скобка!
```

### Пример 3: Ошибка - незакрытые скобки
```
Введите данные: a(b(c)d
Ошибка: не все скобки закрыты! startb=2, endb=1
```

## Принцип работы кода

1. Пользователь вводит строку
2. Программа итерируется по каждому символу строки:
   - При встрече `(` увеличивает счетчик открытых скобок
   - При встрече `)` увеличивает счетчик закрытых скобок и проверяет корректность
   - Для остальных символов проверяет, находятся ли они вне скобок
3. После обработки всей строки проверяется баланс скобок
4. Выводится результат или сообщение об ошибке
