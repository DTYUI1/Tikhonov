# Проверка корректности арифметических выражений

Программа для валидации арифметических выражений, проверяющая синтаксическую корректность с использованием регулярных выражений и логических проверок.

## Описание

Программа анализирует арифметические выражения, проверяя их на соответствие основным синтаксическим правилам. Поддерживаются базовые арифметические операции: сложение, вычитание, умножение, деление, скобки и числа (включая десятичные).

## Проверяемые правила

### 1. Баланс скобок
- Проверяется правильность вложенности и соответствие открывающих и закрывающих скобок
- Пример ошибки: `(2+3))` или `(2+(3)`

### 2. Корректность операторов
- Не допускаются два оператора подряд
- Пример ошибки: `2++3` или `5-*2`
- Проверяется отсутствие оператора в конце выражения
- Пример ошибки: `2+3+`

### 3. Недопустимые символы
- Проверяется, что выражение содержит только разрешенные символы:
  - Цифры `0-9`
  - Операторы `+ - * /`
  - Скобки `( )`
  - Точка для десятичных чисел `.`
  - Пробелы

### 4. Дополнительные проверки
- Проверка на пустые скобки `()`
- Проверка, что выражение не начинается с `*`, `/` или `)`
- Проверка на наличие двух точек в одном числе
- Пример ошибки: `2..5` или `3.14.15`

## Реализованные регулярные выражения

### Основные паттерны:

1. **`([+\-*/]{2,})`** - два или более оператора подряд
2. **`(\.[0-9]*\.)`** - две точки в числе (десятичная запись)
3. **`([+\-*/]\))`** - оператор перед закрывающей скобкой
4. **`^[0-9+\-*/().\s]+$`** - проверка допустимых символов

## Примеры корректных выражений

```
(2+3)*(4-(5/2))+7
3.14 * (2 + 5.5)
10 + 20 - 30 * 2 / 5
((1+2)*3)/4
```

## Примеры некорректных выражений

### Ошибка баланса скобок:
```
(2+3))
((2+3)
```

### Два оператора подряд:
```
2++3
5-*2
```

### Оператор в конце:
```
2+3+
10*
```

### Недопустимые символы:
```
2+3&4
a+b*c
```

### Пустые скобки:
```
2+()+3
```

### Некорректное начало:
```
*2+3
)2+3(
```

## Структура кода

### Основной метод `Main`:
1. Запрашивает у пользователя выражение
2. Вызывает метод `CheckExpression` для проверки
3. Выводит результат проверки

### Метод `CheckExpression`:
Выполняет последовательные проверки:
1. Проверка на пустую строку
2. Проверка баланса скобок
3. Проверка, что выражение не заканчивается оператором
4. Проверка на множественные операторы и некорректные паттерны
5. Проверка, что выражение не начинается с недопустимых символов
6. Проверка на допустимые символы
7. Проверка на пустые скобки
