# RLE String Compressor

Проект реализует алгоритм сжатия строк методом RLE (Run-Length Encoding) и его обратную операцию - декомпрессию.

## Описание

RLE (Run-Length Encoding) - простой алгоритм сжатия данных, который заменяет последовательности одинаковых символов на пару "символ-количество".

**Пример:** 
- `AAABBCCDAA` ? `A3B2C2D1A2`
- `Hello!!!` ? `H1e1l2o1!3`

## Особенности реализации

- Используется `StringBuilder` для эффективной работы со строками
- Не используются регулярные выражения (`Regex`)
- Реализована проверка корректности восстановления
- Подсчет степени сжатия
- Тестовые примеры работы алгоритма

## Требования

- .NET 6.0 или выше
- Компилятор C#
