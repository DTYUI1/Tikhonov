# Filter Collection Project

Проект демонстрирует работу с обобщенными методами и делегатами в C# для фильтрации коллекций.

## Описание

Проект содержит реализацию универсального метода `FilterCollection`, который фильтрует любую коллекцию по заданному условию.

## Основные компоненты

### Класс `Product`
- `name` - название товара
- `price` - цена товара

### Метод `FilterCollection<T>`
- **Параметры**: 
  - `IEnumerable<T> collection` - исходная коллекция
  - `Func<T, bool> condition` - условие фильтрации
- **Возвращает**: `List<T>` - отфильтрованная коллекция

### Метод `More30`
- Фильтрует товары с ценой больше 30 рублей

## Требования

- .NET 6.0 или выше
- Компилятор C#
