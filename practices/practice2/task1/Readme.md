# Versioned Stack Project

Проект реализует версионированный стек с возможностью отслеживания истории изменений.

## Описание

`VersionedStack<T>` - это обобщенная коллекция, которая:
- Хранит элементы в стеке (LIFO)
- Сохраняет историю всех операций Push и Pop
- Позволяет получать состояние по номеру версии

## Требования

- .NET 6.0 или выше
- Visual Studio 2022 или VS Code с расширением C#

## Запуск проекта

### Способ 1: Через Visual Studio

1. Откройте файл `VersionedStackProject.sln` в Visual Studio
2. Нажмите `Ctrl + F5` для запуска без отладки или `F5` для запуска с отладкой

### Способ 2: Через командную строку

```bash
# Перейдите в папку src
cd src

# Компиляция проекта
dotnet build

# Запуск приложения
dotnet run
