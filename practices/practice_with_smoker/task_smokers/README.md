# Лабораторная работа 1: Задача о курильщиках

## Описание задачи
Реализация классической задачи синхронизации "Курильщики" с использованием потоков и мониторов в C#.

## Задача
Есть три курильщика и один агент. Каждый курильщик имеет бесконечный запас одного из трех компонентов:
1. Бумага
2. Табак
3. Спички

Агент случайным образом выкладывает на стол два компонента из трех. Курильщик, у которого есть третий компонент, забирает компоненты со стола, скручивает сигарету и курит. После этого агент выкладывает новые компоненты.

## Технологии и библиотеки
- .NET Core / .NET Framework
- System.Threading
- Мониторы (Monitor) для синхронизации потоков

1. Убедитесь, что у вас установлен .NET SDK
2. Откройте терминал в папке `task_smokers_`
3. Выполните команды:

# Запуск
