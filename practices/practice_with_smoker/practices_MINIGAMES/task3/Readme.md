# Практическая работа 3: Викторина

## Описание задачи
Консольная викторина с вопросами и вариантами ответов. Вопросы загружаются из текстового файла.

## Технологии
- .NET Core / .NET Framework
- Работа с файлами (StreamReader)
- Коллекции (List, массив)

```bash
dotnet new console --force
dotnet build
dotnet run
