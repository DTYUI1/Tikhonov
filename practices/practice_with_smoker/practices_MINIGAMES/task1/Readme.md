# Практическая работа 1: Игра "Угадай число"

## Описание задачи
Простая консольная игра, в которой компьютер загадывает число от 1 до 100, а пользователь пытается его угадать.

## Технологии
- .NET Core / .NET Framework
- Консольное приложение

## Запуск
```bash
dotnet new console --force
dotnet build
dotnet run
